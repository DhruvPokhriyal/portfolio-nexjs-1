## 🤖 Introduction <a name="introduction"></a>

A modern, animated developer portfolio for **Dhruv Pokhriyal** — CS undergrad at **IIT Tirupati**, Full-Stack Developer and AI/ML enthusiast. Built with Next.js for rendering, Three.js for 3D elements, Framer Motion for smooth animations, and Tailwind CSS for styling.

---

## ⚙️ Tech Stack <a name="tech-stack"></a>

- **Framework**: Next.js 14
- **3D**: Three.js
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **UI Components**: Aceternity UI
- **Error Tracking**: Sentry

---

## 🔋 Features <a name="features"></a>

👉 **Hero** — Spotlight effect with animated text generation and a dynamic background grid.

👉 **Bento Grid** — About section with a 3D GitHub globe, tech stack cards, and an email-copy interaction.

👉 **Recent Projects** — 3D pin cards for each project with live links and tech icons.

👉 **Leadership / Experience** — Moving-border cards showcasing positions of responsibility at IIT Tirupati.

👉 **Approach Section** — Canvas-effect animated cards describing development philosophy.

👉 **Footer** — Contact CTA with direct email link and social media icons.

👉 **Responsive** — Fully responsive across mobile, tablet, and desktop.

## 🤸 Quick Start <a name="quick-start"></a>

**Prerequisites**

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en) (v18+)
- [npm](https://www.npmjs.com/)

**Clone the Repository**

```bash
git clone https://github.com/DhruvPokhriyal/portfolio-nexjs-1.git
cd portfolio-nexjs-1
```

**Install Dependencies**

```bash
npm install
```

**Run Locally**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
